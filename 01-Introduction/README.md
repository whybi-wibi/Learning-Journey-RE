# 📘 Section 1 - Introduction

> *"Every Reverse Engineer starts by understanding the fundamentals."*

Selamat datang di **Section 1 - Introduction**.

Section ini merupakan fondasi utama sebelum mempelajari teknik Reverse Engineering yang lebih lanjut. Di sini kita akan memahami konsep dasar, mengenal Assembly, serta mempelajari bagaimana sebuah program dikompilasi menjadi file executable dan bagaimana struktur internalnya.

---

# 🎯 Learning Objectives

Setelah menyelesaikan section ini, Anda diharapkan mampu:

- Memahami konsep Reverse Engineering.
- Menjelaskan perbedaan Forward Engineering dan Reverse Engineering.
- Mengenal dasar teknis Reverse Engineering.
- Memahami dasar bahasa Assembly 64-bit.
- Mengenal struktur file executable.
- Memahami navigasi dasar menggunakan tools Reverse Engineering.

# 📖 Materi

## 1️⃣ Pengantar Reverse Engineering

Mengenal apa itu Reverse Engineering, manfaatnya dalam dunia Cyber Security, serta berbagai bidang yang memanfaatkan teknik ini seperti Malware Analysis, Digital Forensics, dan Vulnerability Research.

📄 **File:** `01-Pengantar-Reverse-Engineering.md`

---

## 2️⃣ Dasar Teknis Reverse Engineering

Mempelajari fondasi teknis Reverse Engineering, termasuk Assembly, Disassembler, Decompiler, dan Debugger sebagai alat utama dalam proses analisis.

📄 **File:** `02-Dasar-Teknis-Reverse-Engineering.md`

---

## 3️⃣ Dasar Assembly 64-bit

Mengenal struktur instruksi Assembly, register, operasi data, serta kontrol alur program yang menjadi dasar dalam membaca hasil disassembly.

📄 **File:** `03-Dasar-Assembly-64-bit.md`

---

## 4️⃣ Struktur Program dan File Biner

Mempelajari struktur file executable seperti Portable Executable (PE), berbagai section penting, Entry Point, serta navigasi dasar menggunakan IDA Free.

📄 **File:** `04-Struktur-Program-dan-File-Biner.md`
