# 📖 About

Reverse Engineering (RE) is the process of analyzing compiled software to understand how it works without access to the original source code.

This repository serves as my personal learning journal where I document concepts, notes, tools, and hands-on exercises while studying Reverse Engineering and its applications in Cyber Security.

The goal is not only to learn how software works internally but also to build a strong foundation for:

- Malware Analysis
- Digital Forensics
- Vulnerability Research
- Software Security
- Windows Internals

---

# 🎯 Objectives

This repository aims to:

- Learn the fundamentals of Reverse Engineering.
- Understand executable file structures.
- Practice static and dynamic analysis.
- Explore debugging techniques.
- Build a foundation for malware analysis.
- Document the learning process for future reference.
